# Event-Hub: Event & Reservation Management System 🚀

This project focuses on the design and implementation of a robust relational database for managing events and bookings.

### Key Features
* **Data Architecture:** Designed a complex relational schema to handle real-time transactions and data consistency.
* **Business Logic & Security (PL/SQL):** Developed advanced **Triggers** to prevent overbooking, ensuring data integrity for both individual and group reservations.
* **Database Automation:** Created SQL scripts for automated data insertion (`InsertionParc`) and structured PL/SQL procedures to streamline workflows.

### Tech Stack
* **Language:** SQL / PL/SQL
* **Concepts:** Triggers, ACID Transactions, Integrity Constraints, Database Automation.
